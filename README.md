# FPGA-GearBox 🚗

## 📋 Project Overview
FPGA-GearBox is an innovative project that combines digital logic design with mechanical engineering. The goal is to develop a 4-speed gearbox controller using an FPGA programmed in VHDL, designed to interface with a custom 3D-printed gearbox model created by team member Gabriel.

This project serves as an extra-curricular spin-off from the course "Fundamentals of Electronics (MOD1: Logic Networks)", demonstrating practical application of digital logic concepts in real-world scenarios.

## 🎯 Features
4-Speed Gear Control - Seamless shifting between 1st, 2nd, 3rd, and 4th gears

FPGA-Based Logic - High-speed, parallel processing capabilities

Custom Hardware Interface - Designed to work with 3D-printed mechanical components

VHDL Implementation - Hardware description language for precise control logic

Real-Time Response - Immediate gear engagement and monitoring

## 🛠️ Technologies Used
Hardware: FPGA Development Board

Language: VHDL (VHSIC Hardware Description Language)

Design Tools: Vivado

Mechanical Components: 3D-printed gearbox model

Version Control: Git/GitHub
